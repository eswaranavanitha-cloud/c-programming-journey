# Day 2 - Variables & Datatypes

## What I learned:

### 1. Variables
Variables are used to store values in memory.

Example:
int age = 20;

---

### 2. Datatypes

- int → whole numbers (10, 20, 100)
- float → decimal numbers (3.14, 5.6)
- char → single character ('A', 'X')

---

### 3. Example:

int a = 10;
float b = 3.14;
char c = 'A';

---

### 4. printf()
Used to display output in C program.

---

## Key Points:
- int = integer values
- float = decimal values
- char = single character
- Variables store data
