# Day 1 - C Programming Basics

## What I learned:

### 1. Structure of C program
Every C program starts with:
#include <stdio.h>
int main() {
    // code
    return 0;
}

### 2. printf()
- Used to display output on screen
- Example:
printf("Hello World");

### 3. main() function
- Program execution starts from main()

### 4. return 0;
- Indicates program ended successfully
