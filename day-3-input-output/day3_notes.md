# Day 3 - Input and Output

## What I learned:

### 1. Input (scanf)
Used to take input from user.

Example:
int num;
scanf("%d", &num);

---

### 2. Output (printf)
Used to display output on screen.

Example:
printf("%d", num);

---

### 3. Important symbol
& → used in scanf to get memory address

---

## Key Points:
- scanf = input
- printf = output
- & is mandatory for variables in scanf
