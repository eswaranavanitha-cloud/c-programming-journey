# Day 5 - If Else (Decision Making)

## What I learned:

### 1. If statement
Used to check a condition.

if (condition) {
    // true block
}

---

### 2. If-Else
Used for decision making.

if (condition) {
    // true block
} else {
    // false block
}

---

### 3. Else-if ladder
Used for multiple conditions.

if (marks >= 90)
    A Grade
else if (marks >= 75)
    B Grade
else if (marks >= 50)
    C Grade
else
    Fail

---

## Key Concepts:

- if = condition true case
- else = condition false case
- else-if = multiple conditions check

---

## Real World Usage:

- Pass/Fail system
- Login validation
- Even/Odd check
- Grading system

---

## Important Rule:

Every condition gives either TRUE or FALSE result.
